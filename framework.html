VV FrameWork

Framework for React Which will simplify creating the folderstructure and file creation along with handling with redux features and api calls

How to install

  • Extract the zip file downloaded from the above download button
  • Go to the Extracted folder in terminal
  • Type ./install-vv
  • You have Successfully installed the frame work

How to use

  • Create a new folder with the project name
  • Open the new folder in terminal.
  • Type vv-new
  • You have Successfully Created your VV Project
  • Now you can edit the vvBase.vv file for components and vvApis.vv for api and redux.
  • Type ./vv-run for generating the react files. Enjoy coding

Tutorial

  • Create a new folder with FirstVV == > mkdir FirstVV
  • Open the new folder in terminal. ==> cd FirstVV
  • Type vv-new
  • You have Successfully Created your VV Project


vvBase.vv

  • The first line of a vvBase.vv file should be Project ProjectName
  • For Creating containers the syntax will be
    "View ScreenName under 'path'"
  • For Creating components the syntax will be
    "Component ComponentName under 'path'"
  • For Creating Routes the syntax will be
    "Route 'routepath' for ScreenName"


vvApis.vv

  • the api will start form api key word
    "api APIName {"
  • The we need to specify the type of request(get/post)
    "type post"
  • Then we will specify the path which is url of the api
    path 'path'
  • Then we will specify the inputs for that api
    input input1,input2
  • Then we will specify the output for that api
    output response


  • After editing this and do a ./vv-run All the files and code for redux and axios will be created.
  • Only thing you need to do is dispatch the api from vvApis.js file and get the response from VVReducer.js file.
  • Happy Coding!!!!!!
